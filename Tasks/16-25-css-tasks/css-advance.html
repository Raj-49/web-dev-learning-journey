CSS Masking Examples

1. image.png - Image appears ONLY in logo (Standard Mask)

White areas in mask = Visible | Transparent/Black = Hidden

2. image-1.png - Image appears in BACKGROUND (Inverted Mask)

Transparent logo cuts out the image, white background shows image

3. image-1.png with filter: invert(1) - Now works like image.png!

Using CSS filter to invert the mask - logo becomes visible area

How to create image.png from logo.png:

Step 1: Open logo.png in an image editor (Photoshop, GIMP, Photopea)
Step 2: Make sure background is transparent (delete white background)
Step 3: Make the logo WHITE color
Step 4: Save as PNG with transparency
Result: White logo on transparent = image shows ONLY in logo

OR use CSS filter: invert(1) on image-1.png to get the same effect!
